Homework 6 - Objects


The task:

Part One A form to see if the user typed a number in the input box has at least four decimal positions.



Part Two A form where the user enters long text and a letter. The letter will then be searched and counted for in the long content, with the text letters being insensitive. If the character is unable to be located, a new window is opened with a message.



Part Three A form in the format of a table that uses a regular expression to determine the appropriate telephone format and return the associated area code, the first three numbers and then the last four digits in the string.
