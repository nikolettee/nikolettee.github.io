Title: Food API

Uncover over thousands of recipes and of food products.



Description

A food API may range from reviewing recipe costs and nutritional values to classifying foods into allergens to computing meal plans, and more! Food and nutrition apps have grown past just being collections of recipes, meal plans and energy values with users looking for personalization--- that is when Food APIs are useful. A recent widely used and popular food API is from Spoonacular. The Spoonacular Food API is an official API that offers SSL support that performs with natural language queries to search for recipes that are according to certain diets and dietary requirements. Specifically, the application calculates nutritional values, analyzes recipe costs, classifies recipes into types and cuisines, and computes an entire meal plan.

Type of response file returned (JSON, XML, etc.): the request format is a URI Query String/CRUD with a response format of a JSON.

Any type of key to access and use the API: requires API Key, OAuth 1, OAuth2, Other/Custom.


While documentation is currently unavailable to provide, some useful features are:

  1. Classify a grocery product
  2. Match recipes to daily calories
  3. Generate meal plan
  4. Classify cuisine
  5. Image analysis and classification
  6. Visualize recipe nutrition, price breakdown, and equipment

Overall, the variety of its beneficial features allows developers to create different kinds of nutrition and food apps.
