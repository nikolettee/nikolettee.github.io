Introduction

Web developers are commonly familiar with three languages:

  1. HTML
  2. CSS
  3. Javascript

HyperText Markup Language (HTML) is the standard markup language to create web pages. The system displays information viewable or audible in web browsers with tags in text files to achieve font, color, graphic, and hyperlink effects on the web! HTML pairs well with Cascading Style Sheets (CSS). CSS is a declarative language used for specifying the layout in HTML documents.

Javascript (JS) is a well-known object-orientated computer programming language commonly used for creating interactive effects on web browsers. Javascript makes HTML pages more dynamic and interactive.

Asynchronous JavaScript And XML (AJAX) uses a combination of a JS and HTML DOM to display or use the data from a browser built-in XMLHttpRequest object that requests data from a web server rather than be called a programming language.
