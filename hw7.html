Homework 7 - Object Model and Cookies


The task:

Part One is a form for the user to choose how to design a few paragraphs on a webpage. The styles that the user selects will be applied immediately and remain until unclicked.



Part Two is a web page with a script that will save two pairs of cookies for the same page - one of the pairs for the name of the user and another one for the username of the user. If it is the first time the user is opening the page on the browser, a form will be presented to type the name of the user and his/her username. If the user has already opened the page and is revisiting the page (while the browser is still opened) a welcome back message will be displayed.
